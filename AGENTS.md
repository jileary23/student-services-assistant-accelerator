# Agent Instructions

This project was built with the microsoft-foundry skill. Before working on or answering questions about Foundry agents, read the microsoft-foundry skill first.

- Keep the deterministic safety router ahead of model calls.
- Do not log message bodies or personal data by default.
- Preserve mock mode so the accelerator remains runnable without Azure resources.
- Use managed identity and Azure RBAC for service-to-service access.
