# Contributing

Thank you for improving the Student Services Assistant Accelerator.

1. Open an issue describing the behavior or documentation change.
2. Create a focused branch and preserve mock mode, the deterministic safety router, and privacy-conscious telemetry.
3. Add or update tests for behavior changes.
4. Run `python -m pytest`, `python -m ruff check .`, and `python -m mypy`.
5. Open a pull request that explains the user impact, safety implications, and validation performed.

Do not include student records, credentials, private institutional content, generated website review bundles, or local Azure environment files. Content examples must be public, synthetic, or explicitly licensed for redistribution.
